# Offloading benchmark

Computation offloading simulation/benchmarking tool implemented as part of my bachelor's thesis: [Energy-efficient computation offloading in Internet of Things devices](https://urn.fi/URN:NBN:fi:aalto-202605234129)

This tool consists of:

- A framework to run configurable and reproducible partial offloading simulations for environments where task generation is stochastic, using different online algorithms for offloading decision
- Plotting the simulation results using `matplotlib`
- Task profiler
